Simulación de Impresora

Sistema operativo que permite registrar usuarios, crear documentos y gestionar una cola de impresión basada en prioridad utilizando un montículo binario.

Incluye tablas de dispersión para ubicar y eliminar documentos en cola de forma eficiente.



\*\*Nota: La prioridad se define en función del tipo de usuario (regular, media, alta), y con respecto al documento, por cantidad de hojas del mismo. El tipo de documento (PDF, DOCX, etc.) se mantiene como un atributo descriptivo, ya que no aporta un criterio funcional válido para ordenar la cola de impresión.
